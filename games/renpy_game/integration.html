The Ethians Redeemed

Ce jeu utilise Ren'Py, un moteur spécialisé dans les visual novels. Une fois compilé, le jeu sera affiché dans le cadre ci-dessous. Pour l'instant, ce n'est qu'une maquette pour montrer comment l'intégration fonctionnera.

Intégration

Pour intégrer votre jeu Ren'Py à votre portfolio, vous devrez :

  1. Créer tous les assets graphiques nécessaires (fonds, personnages, GUI)
  2. Développer votre jeu avec Ren'Py sur votre machine locale
  3. Utiliser l'option "Build Web" dans Ren'Py pour compiler le jeu pour le web
  4. Héberger les fichiers compilés et les référencer dans un iframe comme celui-ci

Une fois compilé, le jeu sera accessible directement depuis votre portfolio avec tous les effets visuels modernes que vous souhaitez (néon, éléments arrondis, etc.).
